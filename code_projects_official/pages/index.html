Para que serve Lógica da Programação ?

A lógica daprogramação também conhecida como lógica computacional, é a forma de como um programa de computador consegue executar comandos através de uma sequência lógica. A criação de um algoritmo nos permite criar e entender o passo a passo para executar as funções necessárias para cada etapa.

Dicas de Sites de Programação:

  • Udemy é uma plataforma de cursos, ela possuir diversos cursos voltados para a programação e outras áreas, os cursos são divididos em diversos módulos na aula. A plataforma possuir cursos tanto em português e inglês.

  • Alura é uma site de cursos, voltada a programação onde você vai estudar, praticar, discutir e se aprofundar em uma plataforma que respira tecnologia.

  • CodeCademy é um site de cursos totalmente voltado para programação, contendo várias linguagens divididas em diversos módulos. Embora boa parte do site esteja em inglês, há uma grande quantidade de conteúdos em português.
